# Grid Overflow


+ Story Show Gallery lightbox
after a click on a thumbnail

.panorama class

Grid Overflow is a CSS library for respon­sive gallery thumbnails layout. It has optional 3D hover effect, utility classes. Adjust­able by CSS variables.

Grid Overflow offers masonry layout and grid layout, where grid items can be given vertigo, pa­no­rama or VIP class to over­flow into the next cell.
